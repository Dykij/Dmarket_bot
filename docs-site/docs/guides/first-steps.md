# 🚀 Первые шаги

После установки и настSwarmки бота, давайте начнем его использовать.

## Запуск бота

ОткSwarmте Telegram и найдите вашего бота по username, который вы указали при создании.

### Команда /start

Отправьте команду `/start` боту. Вы увидите приветственное сообщение и главное меню.

```
Добро пожаловать в DMarket Telegram Bot! 🎮

Я помогу вам автоматизировать торговлю игровыми предметами на DMarket.

Основные возможности:
• 📊 Многоуровневый арбитраж
• 🎯 Система таргетов (Buy Orders)
• 📈 Анализ рынка
• 🔔 Ценовые алерты

Выберите действие:
```

## Главное меню

Вы увидите кнопки:

- **🎯 Арбитраж** - Поиск торговых возможностей
- **🎯 Таргеты** - Управление Buy Orders
- **💰 Баланс** - Проверка баланса DMarket
- **🔔 Алерты** - Ценовые уведомления
- **⚙️ НастSwarmки** - Конфигурация бота

## Первоначальная настSwarmка

### 1. НастSwarmте API ключи

Перейдите в **⚙️ НастSwarmки** → **🔑 API ключи**

Вам нужно будет ввести:
- DMarket Public Key
- DMarket Secret Key

!!! warning "Безопасность"
    Ключи шифруются перед сохранением в базе данных.

### 2. Выберите язык

**⚙️ НастSwarmки** → **🌐 Язык**

Доступные языки:
- 🇷🇺 Русский
- 🇬🇧 English
- 🇪🇸 Español
- 🇩🇪 Deutsch

### 3. НастSwarmте профиль риска

**⚙️ НастSwarmки** → **⚠️ Профиль риска**

Выберите профиль:

- **Консервативный** - Минимальный риск, низкие прибыли
- **Сбалансированный** - Средний риск и прибыль (рекомендуется)
- **Агрессивный** - Высокий риск, высокие прибыли

### 4. Установите лимиты

**⚙️ НастSwarmки** → **💰 Лимиты**

НастSwarmте:
- Максимальную цену для автопокупки
- Максимальное количество активных таргетов
- Дневной лимит трат

## Ваш первый арбитраж

### Шаг 1: Выберите игру

1. Нажмите **🎯 Арбитраж**
2. Нажмите **🎮 Выбор игры**
3. Выберите игру (например, CS:GO)

### Шаг 2: Выберите уровень

Нажмите **📊 Уровни** и выберите уровень:

Для начала рекомендуем:

=== "🚀 Разгон ($0.50-$3)"
    - Низкий риск
    - Быстрая ротация капитала
    - ROI 15-30%
    - Идеально для начинающих

=== "📊 Стандарт ($3-$10)"
    - Средний риск
    - Оптимальный баланс

    - Для опытных пользователей

### Шаг 3: Сканирование

Нажмите **🔍 Сканировать**

Бот начнет поиск арбитражных возможностей. Это может занять 10-30 секунд.

### Шаг 5: Действия

Для каждой возможности вы можете:

- **🛒 Купить** - Купить предмет немедленно

- **❌ Пропустить** - Пропустить эту возможность

## Создание таргета (Buy Order)

### Что такое таргет?

Таргет - это автоматический Buy Order, который:
- Ждет когда цена опустится до указанной
- Автоматически покупает предмет
- Уведомляет вас о покупке

### Как создать?

1. Нажмите **🎯 Таргеты** в главном меню
2. Нажмите **➕ Создать таргет**
3. Введите название предмета
4. Укажите желаемую цену
5. Подтвердите создание

Пример:

```
Название: AK-47 | Redline (Field-Tested)
Цена: $8.00
Текущая цена на рынке: $8.50

✅ Таргет создан!

Бот будет мониторить цену и купит предмет,
когда она опустится до $8.00
```

## НастSwarmка алертов

### Создание ценового алерта

1. Нажмите **🔔 Алерты**
2. Нажмите **➕ Создать алерт**
3. Выберите тип:
   - 📉 Цена ниже
   - 📈 Цена выше
   - 🎯 Точная цена
4. Введите данные предмета и цену

Вы получите уведомление когда условие выполнится.

## Советы для начинающих

!!! tip "Начинайте с малого"
    Рекомендуем начать с уровня "Разгон" ($0.50-$3) для понимания механики.

!!! tip "Проверяйте ликвидность"
    Обращайте внимание на показатель ликвидности. Высокая ликвидность = быстрая продажа.

!!! tip "Не спешите"
    Не обязательно покупать сразу. Создайте таргет и дождитесь лучшей цены.

!!! tip "Используйте DRY_RUN"
    Включите режим DRY_RUN в настSwarmках для тестирования без реальных покупок.

!!! tip "Мониторьте тренды"
    Если тренд "Падающий", возможно стоит подождать с покупкой.

## Часто задаваемые вопросы

### Сколько нужно капитала для старта?

Минимум $50 для уровня "Разгон". Рекомендуем начинать с $100-200.

### Как быстро можно заработать?

Зависит от рынка и уровня. В среднем 5-15% ROI в неделю на уровне "Стандарт".

### Безопасно ли хранить API ключи в боте?

Да, ключи шифруются AES-256 перед сохранением. Только вы имеете доступ.

### Что если цена изменится во время покупки?

Бот проверяет актуальную цену перед покупкой. Если цена выросла, покупка не произойдет.

### Можно ли запускать несколько ботов?

Да, но используйте разные API ключи для каждого бота.
