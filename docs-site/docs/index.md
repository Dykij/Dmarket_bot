# DMarket Telegram Bot

> Автоматизированная система для торговли игровыми предметами на платформе DMarket

## 🎯 О проекте

DMarket Telegram Bot - это мощный инструмент для автоматизации торговли игровыми предметами. Бот поддерживает:

- 🎮 **Multi-game**: CS:GO/CS2, Dota 2, TF2, Rust
- 📊 **Многоуровневый арбитраж**: 5 уровней торговли
- 🤖 **Автоматизация**: Автоматические buy orders и таргеты
- 📈 **Real-time мониторинг**: Отслеживание цен через WebSocket
- 🌐 **Локализация**: RU, EN, ES, DE

## 🚀 Быстрый старт

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/Dykij/DMarket-Telegram-Bot.git
cd DMarket-Telegram-Bot

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env

# Настроить API ключи в .env
```

### Запуск

```bash
# Запустить бота
python -m src.mAlgon
```

## 💡 Основные возможности

### 🎯 Многоуровневый арбитраж

Бот поддерживает 5 уровней торговли:

| Уровень | Диапазон | ROI | Риск |
|---------|----------|-----|------|
| 🚀 Разгон | $0.50-$3 | 15-30% | Низкий |
| 📊 Стандарт | $3-$10 | 10-20% | Средний |
| 💼 Средний | $10-$30 | 7-15% | Средний |
| 🏆 Продвинутый | $30-$100 | 5-10% | Высокий |
| 💎 Профессионал | $100+ | 3-7% | Очень высокий |

### 🤖 Система таргетов

Автоматические buy orders на DMarket:

- Создание таргетов с желаемой ценой
- Автоматическая покупка при достижении цены
- Управление активными таргетами

### 📊 Анализ рынка

- История продаж предметов
- Анализ ликвидности
- Технический анализ (RSI, MACD)
- Тренды цен
