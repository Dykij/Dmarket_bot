# 🗺️ ROADMAP: DMarket Bot Evolution

Этот документ описывает путь развития продукта от текущей версии v3.0 к версии v5.0 "Market Dominator".

---

## 🔴 Critical Fixes (Баги и Стабильность)
*Эти задачи нужно решить в первую очередь для надежности.*

- [ ] **Error Handling 502/504:** Научить бота "засыпать" при ошибках шлюза DMarket, а не падать с крашем.
- [ ] **Reconnect Logic:** Авто-переподключение при разрыве сети (сейчас бот может остановиться).
- [ ] **Balance Sync:** Иногда баланс в кеше может расходиться с реальным. Нужно принудительное обновление перед каждой покупкой.

---

## 🟡 Feature Request (Функционал)
*Инструменты для увеличения прибыли.*

### Phase 1: Network Superiority (Сетевое Превосходство)
- [ ] **HTTP/2 Upgrade:** Миграция клиента с `aiohttp` на `httpx` для поддержки мультиплексирования. Это позволит отправлять пачки запросов параллельно по одному соединению, снижая пинг.
- [ ] **TCP Optimization:** Тонкая настройка Keep-Alive, DNS-кеширования и SSL-контекста. Цель — сделать каждое обращение к API мгновенным (Zero-Latency Handshake).
- [ ] **Adaptive Polling:** Умный алгоритм, который подстраивает частоту запросов под нагрузку сервера DMarket, чтобы избегать ошибок 429 (Rate Limit) без использования прокси.
- [ ] **Notification Filters:** Настройка в Telegram: "Присылай уведомления только если профит > $1".

### Phase 2: Deep Value (Глубокая Аналитика)
- [ ] **Sticker Valuation:** Подключение внешнего API (Buff163/CSFloat) для оценки наклеек. (Купить за $10 -> Продать за $50 из-за Kato14).
- [ ] **Paint Seed Analysis:** Поиск редких паттернов (Blue Gem, Fire & Ice).
- [ ] **Database History:** Переход от JSON логов к полноценной базе данных (SQLite/PostgreSQL) для аналитики "Какой скин самый прибыльный за месяц?".

---

## 🟢 Architecture (Архитектура)
*Техническое совершенство.*

- [ ] **Dockerization:** Упаковать бота в Docker-контейнер. Это позволит разворачивать его на любом VPS одной командой.
- [ ] **Redis Queue:** Разделить сканирование и торговлю. Сканнер кидает задачи в очередь, Трейдер их исполняет. Это повысит скорость реакции в 10 раз.
- [ ] **Web Dashboard:** Полноценная веб-панель с графиками вместо Telegram-сообщений.
