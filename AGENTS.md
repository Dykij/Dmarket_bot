# AGENTS.md — Инструкции для Аркадия

## Стэк проекта
- Язык: Python 3.12+
- Библиотека: aiogram 3.x (используй только актуальные методы)
- Данные: Pydantic v2 для моделей, Pandera для валидации таблиц
- Логи: loguru

## Правила кодинга
1. Humanizer: Избегай "AI-вайба". Пиши код и комментарии просто и по делу. Не используй слова "надежный", "комплексный", "бесшовный".
2. Context Hygiene: Если файл больше 500 строк, предлагай разбить его на подмодули.
3. Error Handling: Оборачивай запросы к API DMarket в try-except с логированием через loguru.
4. Secrets: Никогда не хардкодь токены. Используй .env и os.getenv.

## Структура папок
- /handlers — логика команд бота
- /keyboards — кнопки (Reply/Inline)
- /services — работа с API DMarket
- /database — работа с БД или файлами хранения
