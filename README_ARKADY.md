# 🤖 Arkady Jarvis: Инструкция по управлению

Добро пожаловать в систему управления Аркадием. Теперь у тебя есть "пульт управления" прямо в консоли PowerShell.

## 🚀 Активация среды
Чтобы активировать все команды Аркадия в текущем окне терминала, выполни:
```powershell
. .\setup_arkady_env.ps1
```

## 🎮 Пульт управления (Алиасы)
После активации тебе доступны следующие команды:

| Команда | Описание |
| :--- | :--- |
| `ark-logs` | Показать последние 20 действий Аркадия (activity.jsonl) |
| `ark-status` | Вывести текущую фазу эволюции и статус системы |
| `ark-watch` | Запустить мониторинг GitHub Actions (в реальном времени) |
| `ark-report` | Открыть свежий аналитический отчет в браузере |

## 🛠 Технический стек
- **Runtime**: Docker (Python 3.12-slim)
- **CI/CD**: GitHub Actions + GitHub CLI
- **Intelligence**: Gemini 3 Flash (v2 Wrapper with Caching)
- **Monitoring**: Arkady Watcher 2.0
